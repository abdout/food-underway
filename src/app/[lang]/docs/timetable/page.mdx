### üöÄ Getting Started

#### Initial Setup

1) Navigate to **Timetable** from the main menu
2) Click **Settings** (gear icon) to configure:
   - Working days (e.g., Mon-Fri, Sun-Thu)
   - Periods per day and duration
   - Lunch and break periods
   - Color scheme preference
3) Import existing data or start creating slots manually

### üìù Managing Timetable Slots

#### Adding New Slots

1) **Click the "+ Add Slot" button** or click any empty cell in the grid
2) Fill in the required information:
   - Day and Period
   - Class/Section
   - Subject (with automatic color coding)
   - Teacher (filtered by subject expertise)
   - Classroom (shows availability)
3) **Optional**: Enable recurring to create the same slot for multiple weeks
4) Click **Save** to add the slot

#### Editing Existing Slots

**Method 1: Drag & Drop**
- Simply drag any slot to a new time/day position
- The system validates conflicts automatically
- Visual feedback shows valid drop zones

**Method 2: Edit Dialog**
- Click the menu icon (‚ãÆ) on any slot
- Select **Edit** to modify details
- Change teacher, room, or add substitute

#### Bulk Operations

- **Duplicate**: Copy slots to other days/periods
- **Move Multiple**: Select multiple slots with Ctrl/Cmd+Click
- **Delete Multiple**: Select and press Delete key

### ‚ö†Ô∏è Conflict Detection & Resolution

The system automatically detects three types of conflicts:

| Conflict Type | Description | Resolution |
|--------------|-------------|------------|
| **Teacher Conflict** | Same teacher in multiple classes | Reassign teacher or adjust timing |
| **Room Conflict** | Room double-booked | Find alternative room or time |
| **Class Conflict** | Class scheduled twice in same period | Remove duplicate slot |

#### Resolution Workflow

1) **Automatic Detection**: Conflicts appear as a red badge in the header
2) **View Details**: Click the conflict counter to see all conflicts
3) **Smart Suggestions**: System suggests alternative:
   - Available time slots
   - Free teachers with same subject expertise
   - Unoccupied classrooms
4) **One-Click Fix**: Apply suggestions instantly
